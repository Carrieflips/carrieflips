Build Log: Focacciadoro · Web · April 2026
Web · In Development

A web app that combines the rhythm of making focaccia with intervals of focused work called pomodoros.

Build Log

Apr 28

First working timer

Core pomodoro interval running — 25 min focus, 5 min rest. The bread metaphor is clicking. Each break is a "fold." Each long rest is "proofing."

Apr 21

Concept and naming

Focacciadoro = focaccia + pomodoro. The idea: structure your work around the rhythm of making bread. Started with plain HTML and vanilla JS — no framework, no build step.

Apr 14

The inspiration

Made focaccia on a Sunday and kept checking the clock. The resting intervals mapped perfectly to pomodoro breaks. Wanted something that made the parallel explicit.
